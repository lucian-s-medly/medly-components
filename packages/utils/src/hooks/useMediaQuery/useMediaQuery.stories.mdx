# useMediaQuery

The `useMediaQuery` hook checks if the current screen satisfies the passed media query.

```tsx
const matches = useMediaQuery('@media (min-width: 768px) and (max-width: 1240px)');
```

You can use the `breakpoints` function to get the desired media query like below:

```tsx
import { breakpoints, ThemeContext, useMediaQuery } from '@medly-components/utils';
import React, { useContext } from 'react';

export const DummyComponent: React.FC = React.memo(() => {
    const theme = useContext(ThemeContext),
        matches = useMediaQuery(breakpoints(theme.breakpoints).up('M'));

    return {matches ? <>'Matched'</> : null};
});
```

> Note: If you use typed styled-components, import `ThemeContext` from the exported styled file in your project.
