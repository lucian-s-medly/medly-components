# useCombinedRefs

You can use the `useCombinedRefs` hook to combine multiple refs. For example, while developing low-level UI components, it is common to have a local `ref` but support an external one as well. Since React does not offer a way to set multiple refs inside `ref` property, use this hook to combine multiple refs.


#### Example

```tsx
import { useCombinedRefs } from '@medly-components/utils';
import { Text } from '@medly-components/core';

export const Demo = ({ someOutsideRef }) => {
    const combinedRefs = useCombinedRefs<HTMLInputElement>(someOutsideRef, useRef(null));

    return <input ref={combinedRefs} type="text" />;
};
```
