# useWindowSize

You can use the `useWindowSize` to get the current size of the browser window. It returns an object containing the window's width and height.

#### Example

```tsx
import { useWindowSize } from '@medly-components/utils';
import { Text } from '@medly-components/core';

export const Demo = () => {
    const { height, width } = useWindowSize();

    return (
        <>
            <Text>The current window dimensions are: </Text>
            <code>{JSON.stringify({ height, width })}</code>
        </>
    );
};
```

**Note: It uses debouncing internally, to improve performance.**
