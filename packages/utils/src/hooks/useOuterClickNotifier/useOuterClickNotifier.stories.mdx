# useOuterClickNotifier

The `useOuterClickNotifier` hook triggers the provided function on certain events affecting the `ref` passed.

More specifically, the function triggers when:

- A click occurs anywhere outside the ref target.
- Whenever you press the <kbd>ESC</kbd> key.

## Parameters

- The `onOuterClick` is a function to be triggered.
- The `innerRef` is a ref to some element.

### Example

The following example demonstrates a simple use of this hook to clear the contents of
a text input after you finish typing in it and meet the trigger conditions
outlined above.

```tsx
const AutoClearTextInput: React.FC = () => {
    const ref = useRef(null);
    useOuterClickNotifier(() => {
        if (ref.current) {
            ref.current.value = '';
        }
    }, ref);
    return <input ref={ref} type='text' />;
};
```
